## Chrome Web Store

### Short Description (132 chars max)
A beautiful, dark-mode messaging client for Bluesky. Send DMs, react with emojis, and hide engagement metrics with Zen Mode.

### Detailed Description
Bluedrop brings Bluesky Direct Messages to your browser toolbar in a sleek, privacy-focused extension.

FEATURES:
- Native Dark Mode matching Bluesky's aesthetic
- Real-time messaging with instant delivery
- Emoji reactions with a single click
- Message request handling (accept/decline)
- Unread count badges on the extension icon

*ZEN MODE*
Hide likes, replies, reposts, quotes, and follower counts on bsky.app for a distraction-free experience. Reduce social comparison and enjoy a calmer feed.

PRIVACY:
Uses Bluesky App Passwords for secure authentication. Your credentials are stored locally and never leave the extension. No tracking, no analytics, no data collection.

Connects directly to the official Bluesky API. Open source.

## Mozilla Firefox Add-ons

### Summary (250 chars max)
Dark-mode messaging for Bluesky. Send DMs, react with emojis, and enable Zen Mode to hide engagement metrics on bsky.app. Privacy-first: uses App Passwords, no tracking.

### Description

Bluedrop is a lightweight, privacy-focused messaging extension for Bluesky users who want quick access to their Direct Messages without keeping a tab open.

WHY BLUEDROP?
- Beautiful Design - Matches Bluesky's native dark mode with vibrant blue accents
- Fast and Responsive - Real-time messaging with instant updates
- Privacy First - Uses App Passwords. No tracking. No analytics. Your data stays yours.

*ZEN MODE - THE HIGHLIGHT FEATURE*
Take control of your feed. Hide likes, replies, reposts, quotes, and follower counts on bsky.app. Reduce social comparison and enjoy a calmer browsing experience.

KEY FEATURES:
- Dark Mode - Fully themed to match Bluesky's official aesthetic
- Emoji Reactions - React to any message with your favorite emojis
- Profile Previews - Tap any avatar to see their profile inline
- Message Requests - Accept or decline new conversation requests
- Zen Mode - Hide engagement metrics on bsky.app
- Keyboard Shortcuts - Cmd/Ctrl+K for new chat, Esc to go back

PRIVACY AND PERMISSIONS:

Bluedrop connects directly to the Bluesky API (bsky.social) and the official chat proxy. Only the permissions necessary are requested:
- Storage: To save your session locally
- Notifications: For unread message alerts
- Host permissions: To communicate with Bluesky servers

No data is collected, tracked, or shared. This extension is open source.
