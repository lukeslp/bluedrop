# Bluedrop for Bluesky

[![Version](https://img.shields.io/badge/version-1.0.5-blue.svg)](https://github.com/lukeslp/bluedrop/releases)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

A dark-mode messaging extension for Bluesky. Send DMs, react with emojis, and hide engagement metrics with Zen Mode — all from your browser toolbar.

---

## Features

- **Dark Mode** — Matches Bluesky's native aesthetic with vibrant blue accents
- **Real-time messaging** — Instant delivery with unread count badges on the extension icon
- **Emoji reactions** — React to any message with a single click
- **Message requests** — Accept or decline new conversations inline
- **Profile previews** — Tap any avatar to see a profile card without leaving the chat
- **Keyboard shortcuts** — `Cmd/Ctrl+K` for new chat, `Esc` to go back

### Zen Mode

Hide likes, replies, reposts, quotes, and follower counts on bsky.app. Cuts the noise for a calmer, less comparison-driven feed.

### Firefox
Available on [Firefox Add-ons](https://addons.mozilla.org) — search "Bluedrop".

---

## Privacy

Uses Bluesky App Passwords for authentication. Your credentials stay local — the extension never phones home, collects data, or runs analytics. Connects directly to the official Bluesky API (`bsky.social`).

Permissions requested:
- **Storage** — saves your session locally
- **Notifications** — unread message alerts
- **Host permissions** — communicate with Bluesky servers only
